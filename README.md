# ELF EXPRESS 集運中心

現代化的集運管理儀表板，支援多主題切換功能。

## 🎨 主題系統

應用程式支援 **6 種精美的顏色主題**：

- 🔵 **經典藍** - 專業沉穩的藍色系（預設）
- 💜 **夢幻紫** - 優雅浪漫的紫色系
- 💚 **清新綠** - 自然舒適的綠色系
- 🧡 **活力橙** - 熱情活潑的橙色系
- 💗 **浪漫粉** - 溫柔甜美的粉紅系
- 🩵 **科技青** - 現代科技的青色系

### 如何切換主題

1. 點擊頂部導航欄右側的 **調色盤圖標** 🎨
2. 從下拉選單中選擇您喜歡的主題
3. 主題會立即應用並自動保存到本地儲存

### 主題特色

- ✨ 即時切換，無需刷新頁面
- 💾 自動保存選擇，下次訪問時保持
- 🎯 全局主題色統一，包括：
  - 按鈕和交互元素
  - 用戶卡片的漸變背景
  - 進度條和狀態指示
  - 表單輸入框焦點狀態

## 🚀 快速開始

### 安裝依賴
```bash
npm install
```

### 開發模式
```bash
npm run dev
```

應用將在 `http://localhost:5173` 啟動

### 建置生產版本
```bash
npm run build
```

## 🛠 技術棧

- **React 18** - UI 框架
- **TypeScript** - 類型安全
- **Vite** - 快速建置工具
- **Tailwind CSS** - 原子化 CSS
- **shadcn/ui** - 高質量 UI 組件
- **Framer Motion** - 流暢動畫
- **Recharts** - 數據圖表
- **Lucide React** - 現代圖標庫

## 📁 專案結構

```
src/

│   ├── ui/              # 基礎 UI 組件
│   ├── ConsolidationDashboard.tsx  # 主儀表板
│   └── ThemeSwitcher.tsx          # 主題切換器
├── contexts/
│   └── ThemeContext.tsx  # 主題上下文管理
├── lib/
│   └── utils.ts         # 工具函數
├── App.tsx              # 應用入口
├── main.tsx            # React 入口
```
